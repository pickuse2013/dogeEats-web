## 關於DogeEats

DogeEats是一個學校專題專案，目的在於練習開發Android App專案，本專案屬於其底下的網站功能，負責提供API連接資料庫、處理商業邏輯、同步資訊等。

本專案使用Laravel框架進行開發，請至安裝目錄輸入以下指令來進行自動化安裝：

``
composer install
``
